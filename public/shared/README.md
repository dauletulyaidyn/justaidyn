# Public Shared Files

Files in this folder are intended for public read-only access by direct URL.

Use this path for images that anyone can open or download by link:

`/static/shared/images/file-name.png`

The repository also keeps the same public files under `static/shared/` because
the production nginx server serves `/static/` directly from that folder.

The same files are also available locally through:

`/public/shared/images/file-name.png`

Do not place private, paid, internal, or personal documents here.
