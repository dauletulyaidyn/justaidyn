Built for real recording work

One app for teaching, explaining, and publishing faster.

ScreenCam keeps the common recording workflow compact: prepare the screen, show yourself when needed, annotate while recording, then save the video and subtitle file.

Online lessons

Record lectures, homework explanations, coding walkthroughs, and student feedback with a clear camera presence.

Product demos

Capture product flows with custom regions, pause/resume hotkeys, and a watermark for branded delivery.

Short-form content

Use presets for social formats and export MP4 recordings that are easier to reuse across channels.

Team explanations

Create quick internal videos for support, onboarding, bug reports, and async collaboration.

Feature set

Everything important stays close during the session.

Region-based screen recording

Select a capture area, use common presets, and save recordings as MP4 files.

Camera preview and rotation

Keep a live camera window on top, choose cameras, and rotate between selected devices.

Audio capture

Record microphone and optional computer audio for lessons, calls, and demonstrations.

Pointer and drawing tools

Use cursor, laser, pen, highlighter, drawing layers, and quick cleanup controls while recording.

Watermark controls

Add watermark text, choose one of nine positions, and control opacity in the live preview and output.

Subtitle file generation

Generate an SRT subtitle file automatically after recording stops.

Workflow

From setup to final file in five steps.

1

Sign in

Launch the app and use Google sign-in before the main screen opens.

2

Prepare

Choose cameras, recording region, audio devices, watermark, hotkeys, and save location.

3

Record

Start the session and keep the live camera preview visible while you work.

4

Annotate

Use pointer, laser, pen, and highlighter tools when the explanation needs attention.

5

Export

Stop recording and save the MP4 output with a generated subtitle file.
