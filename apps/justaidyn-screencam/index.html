JustAidyn ScreenCam

Compact Electron app that combines a live rotating camera preview with configurable screen recording.

Download for Windows All Apps
Capabilities

What It Does

  • Live camera preview in a small always-on-top window
  • Pick which cameras participate in preview and rotation
  • Separate frameless setup window before start
  • Screen recording with selectable capture region
  • Records to MP4 format
  • Optional computer audio and microphone capture
  • Custom recording hotkeys for stop and pause/resume
  • Watermark text with 9 anchor positions and opacity control
  • Floating pointer menu: Cursor, Laser, Pen, Highlighter
  • Drawing layers with create, navigate, hide, delete, clear-all
  • Camera background effects: Blur or Background on one selected camera
  • Automatic .srt subtitle file generated after recording stops
  • Google sign-in before opening the main app
Workflow

How It Works

  1. 1 Launch the app and sign in with Google.
  2. 2 The preview window opens and shows the first selected camera.
  3. 3 The setup window opens separately — configure cameras, recording region, audio, watermark, hotkeys, and save location.
  4. 4 Click Start. The setup window closes and the live session begins.
  5. 5 When recording stops, the setup window opens again and an .srt subtitle file is generated automatically.
All Features

Main Features

  • Borderless preview and setup windows
  • Google authentication gate before the app opens
  • Camera rotation interval selection and order management
  • Horizontal and vertical preview flip
  • Restore preview window position and size on launch
  • Capture presets: HD, Full HD, QHD, Instagram, WhatsApp, YouTube, TikTok…
  • Custom capture-region selector with move and resize handles
  • Optional system audio and single microphone selection
  • Save folder preselection or prompt after stop
  • Watermark text with 9 placement options and opacity 30–50%
  • Global hotkeys for recording stop, pause, and resume
  • Single-camera background effects: Blur and Background 1
  • Right-click camera switching during recording
  • Single-instance protection to block parallel launches
  • Floating live capture preview window with watermark overlay
  • Automatic subtitle (.srt) generation after recording
